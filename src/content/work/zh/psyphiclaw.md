## 问题

传统的行为数据分析很慢——有时编码并分析一个数据集需要数周。作为多年从事行为编码的从业者，我提出一个问题：如果我们能用多智能体 AI 系统把它自动化呢？

## 技术架构

PsyPhiClaw 基于 Python 构建，使用 LLM API 作为推理引擎，通过 RAG 检索行为编码知识库来约束输出。系统采用多智能体分工：动作追踪智能体处理视频/观察数据，biosignal 智能体处理 EEG/GSR 等生理数据，模式识别智能体做跨模态融合分析——各自专注一个维度，再由协调智能体汇总成结构化洞察。

## 当前阶段

平台处于概念验证（PoC）阶段，已验证核心假设：多智能体协作确实能将原本数小时的人工编码工作显著压缩。当前局限在于数据源覆盖与一致性校验，下一步规划是接入更多行为数据类型，并向生产化演进。

## 信念

它承载着我的信念：行为科学的未来，在于人类专长与 AI 能力的交汇——AI 不取代人，而是放大人类分析能力的杠杆。这也是我对 Noldus 行为数据资产未来价值的核心判断。
