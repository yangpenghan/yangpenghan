## 领域交叉价值（核心差异化）

```
        行为分析技术
        （Noldus 产品生态）
           ▲
          / \
         /   \
  医疗器械      AI Agent
  可用性工程     应用开发
  （IEC 62366） （多智能体）
```

### 为什么这个交叉稀缺？
1. **懂行为学的人不多**：全球 Noldus 认证/深度使用者可能只有几千人
2. **懂可用性工程的更少**：在中国能独立完成医疗器械可用性验证的人极少
3. **同时懂 AI 的是凤毛麟角**：能将三者结合应用的人可能全球不超过几百
4. **商业价值**：AI + 行为分析 = 新一代智能行为洞察；AI + 可用性 = 自动化人因测试
